# Send-to-thread
フォーラムチャンネルに新しいスレッドが作成されたときにメッセージを送信するBOT

## 実行手順
1: [.env](https://github.com/VEDA00133912/Send-to-thread/blob/main/.env)に自分のBOTのTokenを入力する
2: node index.jsで実行
3: 実際にフォーラムチャンネルに投稿を作成して画像のようになるかを確認
